# lsshop-albertmiranda

Proyecto web desarrollado con **Laravel (PHP)** como práctica académica. Incluye estructura MVC completa (rutas, controladores, vistas y acceso a datos).

## Stack
- **Backend:** PHP + Laravel
- **Frontend (assets):** Vite (según configuración del proyecto)
- **BBDD:** MySQL/MariaDB (o la que configures en `.env`)

## Requisitos
- PHP 
- Composer
- Node.js + npm
- Base de datos (MySQL/MariaDB recomendado)
